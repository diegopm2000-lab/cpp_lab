# Calculator in C++

Calculator implementation in C++

The following functions are implemented:

- add
- substract
- multiply
- divide

### 1. Compile the program

Execute:

```shell
$ g++ *.cpp -o calculator.out
```

This generates a name executable __calculator.out__

### 2. Execution of the program

Execute

```shell
$ ./calculator.out
a + b = 106.4
a - b = -91.6
a * b = 732.6
a / b = 0.0747475
```
